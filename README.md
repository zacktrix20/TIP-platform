# Threat Intelligence Platform (TIP)

A sophisticated, centralized platform for the aggregation, analysis, and visualization of open-source threat intelligence (OSINT). Designed for security operations centers (SOC) and incident response teams.

## Key Features

- **Real-Time Data Ingestion:** Automated aggregation from multiple OSINT sources.
- **Advanced Analytics:** AI-powered deduplication and risk scoring.
- **Visual Intelligence:** Interactive network topology and global threat maps.
- **Mission Console:** A high-fidelity dashboard for comprehensive threat monitoring.
- **Analyst Lab:** Specialized tools for deep-dive indicator investigation.

## Tech Stack

- **Frontend:** React, TypeScript, Tailwind CSS, Framer Motion, Recharts, D3.js.
- **Backend:** Node.js, Express.
- **Database:** Firestore (Enterprise Edition).
- **AI Service:** Integrated Intelligence Sublayer.

## Security & Compliance

- Encrypted communications.
- Attribute-Based Access Control (ABAC).
- Hardened database security rules.
